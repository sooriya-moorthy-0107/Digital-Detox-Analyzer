# Digital Detox Analyzer

A complete web application to track and analyze your screen time habits.

## How to Run

Simply open `index.html` in any web browser (Chrome, Firefox, Edge, etc.)

**No installation or server required!**

## Features
- Login/Signup system
- Dashboard with interactive charts
- Log screen time with mood, productivity & health tracking
- Analysis with insights & recommendations
- Personalized detox plan generator
- CSV data export
- Fully responsive design

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Chart.js for data visualization
- Font Awesome icons
- Google Fonts (Poppins)
- LocalStorage for data persistence
